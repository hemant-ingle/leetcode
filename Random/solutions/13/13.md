## Binary Search Invariant (Lower Bound)
**Invariant:** `l < answer <= r`, where `nums[l] < target` and `nums[r] >= target` always holds.
Initially `l = -∞` (`-1`), `r = +∞` (`n`).

**Time Complexity:** *O(logn),* **Space Complexity:** *O(1).*

```cpp
    class Solution {
    public:
        int search(vector<int>& nums, int target) {
            int l, m, r;
            l = -1;
            r = nums.size();
            while(r > l + 1) {
                m = l + (r - l) / 2;
                if(nums[m] < target) {
                    l = m;
                } else {
                    r = m;
                }
            }
            if(r < nums.size() && nums[r] == target) return r;
            return -1;
        }
    };
```
